# Jafuta Foundation Platform
Built by Mandla Ncube

A digital management platform built for Jafuta Foundation, combining two programmes into one web app.

💡 Innovation Hub
- Visitor Check-In System (iPad kiosk)
- Lesson Plan Builder (Primary / Secondary / MCRI)
- Admin Dashboard with CSV export

📚 English Literacy Programme
- Daily student attendance register
- Student enrolment (100+ learners)
- Reading progress tracker (Pre-Reader / Early Reader / Fluent)
- Teacher reports with CSV export

 🛠 Built With
- HTML, CSS, JavaScript
- Hosted on GitHub Pages
